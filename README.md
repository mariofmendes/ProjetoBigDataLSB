O Projeto Big Data LSB foi desenvolvido como trabalho de extensão para a matéria Tópicos de Big Data em Python. 
O tema escolhido foi a LSB - Liga Super Basketball, uma liga que organiza jogos de basquete de várias equipes e várias modalidades. As equipes são tanto do Rio de Janeiro como de outros estados, mas a essência da LSB são as equipes da Baixada Fluminense do Rio de Janeiro.
O projeto dedicou-se a analisar a base de dados que foi coletada sobre toda a série histórica disponível, de todas as equipes que participaram e as modalidades criadas. Foi feita a análise e criado um dashboard com as conclusões acerca dessa base de dados.

O projeto foi desenvolvido na linguagem de programação python, utilizando as bibliotecas: streamlit, pandas, plotly. 
Todo o código foi estruturado no padrão de arquitetura de software, MVC, o que permite dividir uma aplicação em três camadas interconectadas para separar responsabilidades. 
Como IDE para o desenvolvimento do projeto, foi utilizado o VS Code e a base de dados foi estruturada em CSV.
